## ğŸš€ Getting Started

### Requirements

- JDK 8+ (for Java parts)  
- Python 3.7+ (for scripts in `imdb_python`)  
- An IDE such as IntelliJ or Android Studio

### Clone the repo

```bash
git clone https://github.com/AaronCreor/FriendFlicks.git
cd FriendFlicks
```

### Running

#### Java Application

Open the project in your IDE and build/run the main application entry point, or from the command line:

```bash
./gradlew build
```

Then run the generated artifact or the main class from your IDE.

#### Python Scripts

```bash
cd imdb_python
python3 script_name.py
```

(Replace `script_name.py` with the script you want to run.)
